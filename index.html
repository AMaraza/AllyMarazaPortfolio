Who Am I

I’m a Front-End Web Developer, Tabletop Game Designer, and C++ Programmer who enjoys building practical, hands-on projects.

I started in game design with Unreal Engine and Blueprints, where I learned to create interactive systems and dynamic experiences. Now, I’m working toward becoming a full-stack developer—combining my coding knowledge with user-focused design to build responsive websites and applications.

One of my top achievements is successfully launching a tabletop game on Kickstarter. That process taught me a lot about project management, creative thinking, and what it takes to turn an idea into something real.

Dun-Dun-Dungeon


This project was a part of a Systems Design class where over a month we had to design a game using only pieces players could find around their house, and it had to fit on one page of paper. One side for the rules, one side for the board.

During this project, we prototyped, gathered playtesting data, calculated probability, and tracked player turns. As an extension of the project, I created multiple ways to play the game fitting into the same restrictions for the expansions.

Meeple Ball


This project took a digital game that a group of game designers and I made for a class assignment, and turned it into a tabletop strategy board game. We had to stick close to the mechanics of our reference game.

I was tasked with creating the rules, and implemented player movement similar to chess where each piece moves in certain directions for a certain number of spaces. I also designed a stamina system to represent the digital counterpart where stamina was exhausted by dashing.
