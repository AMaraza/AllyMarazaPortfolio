Livewire was a game made in Python for a week long game jam, with the theme of "Keep it Alive". The game has you maintain power by placing batteries and connecting traces to power draining cores.

The entire game was written in python using the Pygame library. This was my first major pygame project, and learning how to do a tile based game like this was a learning curve. It was also one of my first experiences with object oriented programming and multi file programming within python.

To put the game onto Itch.io for the submission for the game jam, I used a tool called Pygbag, which takes python projects and packages them into web assembly so that they can run directly in web browsers.
